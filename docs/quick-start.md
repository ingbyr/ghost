# Ghost Host Manager - 快速入门指南

## 安装与运行

### Windows
1. 下载并运行 Ghost Host Manager
2. **首次运行时**：Windows UAC 对话框将弹出，询问是否允许应用以管理员身份运行
3. 点击"是"或输入管理员密码以授予所需权限
4. 应用将以管理员权限启动，后续操作无需重复认证

### Linux
1. 下载并运行 Ghost Host Manager
2. **首次需要修改hosts时**：图形化密码输入框将弹出
3. 输入您的用户密码以授权应用修改系统文件
4. 在 sudo 缓存有效期内，后续操作无需重复认证（通常为15分钟）

### macOS
1. 下载并运行 Ghost Host Manager
2. **首次需要修改hosts时**：系统密码输入对话框将弹出
3. 输入您的管理员密码以授权应用修改系统文件
4. 在 sudo 缓存有效期内，后续操作无需重复认证（通常为5分钟）

## 为什么需要管理员权限？

Ghost Host Manager 需要修改系统关键文件 `/etc/hosts`（Linux/macOS）或 `C:\Windows\System32\drivers\etc\hosts`（Windows）。这些文件受到操作系统保护，只有具有管理员权限的程序才能修改它们。

## 界面语言设置

### 语言支持
- Ghost Host Manager 支持中文和英文两种界面语言
- 系统会自动检测您的浏览器语言偏好

### 手动切换语言
1. 在应用界面右上角找到语言切换下拉菜单
2. 点击下拉箭头选择您偏好的语言
3. 选择后界面语言会立即切换，无需重启应用
4. 您的语言偏好将被保存，下次启动时会自动使用该语言

## 权限安全说明

- 应用仅在需要修改 hosts 文件时请求权限
- 不会获取超出修改 hosts 文件所需的权限
- 遵循最小权限原则，专注于 hosts 管理功能
- 所有权限请求都是透明的，用户可以清楚地看到何时需要授权

## 常见问题

### Windows
**Q: 为什么每次运行都弹出UAC对话框？**
A: 请检查您的账户是否具有管理员权限。如果问题持续存在，请以管理员身份右键点击应用图标，选择"属性" -> "兼容性" -> 勾选"以管理员身份运行此程序"。

### Linux
**Q: 没有看到图形化密码输入框？**
A: 您的系统可能缺少图形化 sudo 工具。请安装以下工具之一：
- Ubuntu/GNOME: `sudo apt install policykit-kde-agent` 或 `pkexec`
- KDE: `sudo apt install kde-sudo` 或 `kdesudo`
- 通用: `sudo apt install gksu` 或 `gksudo`

### macOS
**Q: 密码输入后仍然无法修改hosts文件？**
A: 请确保您的用户账户具有管理员权限。您可以在"系统偏好设置" -> "用户与群组"中检查。

## 故障排除

如果您遇到权限相关问题，请检查：
1. 您的用户账户是否具有管理员/超级用户权限
2. 系统防火墙或安全软件是否阻止了应用的网络访问
3. 是否有其他程序正在锁定 hosts 文件

更多信息请参阅完整[权限管理说明](./permissions.md)。
